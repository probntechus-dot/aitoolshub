4. Attribution: "Photo by [Photographer Name] on Unsplash"

### Using Pexels:

## Image Optimization Workflow

After downloading:
1. Resize to target dimensions (ImageMagick or online tool)
2. Compress to target file size (TinyPNG or similar)
3. Save with descriptive filename
4. Upload to `/images/` directory
5. Update HTML with new local path
6. Add attribution in article footer
